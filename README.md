# VanillaJS

[Official Documentation](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

```bash
npm install && npm start
```

## Features

* offline capability
* one-way data binding
* uri-based routing
* native app experience
* custom web components
* classes
* module loading
* string templates
* async HTTP calls
* DOM element retrieval
